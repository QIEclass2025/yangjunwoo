# 포켓몬 행맨 (Pokémon Hangman)

1세대 포켓몬 151마리 중에서 **무작위로 1마리를 선택**하여,  
그 포켓몬의 이름을 맞히는 **행맨(Hangman) 게임**입니다.  

GUI 환경에서 포켓몬 이미지와 함께 글자를 추리하며 플레이할 수 있습니다.

---

## 1. 게임 설명

- **게임 이름**: 포켓몬 행맨 (Pokémon Hangman)
- **플레이 방식**
  - 게임 시작 시, 1세대 포켓몬 151마리 중에서 1마리가 무작위로 선택됩니다.
  - 선택된 포켓몬의 이름이 숨겨진 상태(밑줄 `_`)로 표시됩니다.
  - 알파벳 버튼(또는 키보드 입력)을 통해 글자를 하나씩 추리합니다.
  - 맞는 글자를 선택하면 해당 위치에 글자가 공개되고,  
    틀리면 목숨(시도 횟수)이 줄어듭니다.
- **승리 조건**
  - 제한 횟수 내에 포켓몬 이름의 모든 글자를 맞히면 승리합니다.
- **패배 조건**
  - 제한 횟수를 모두 소진했는데도 아직 맞히지 못한 글자가 남아 있으면 패배합니다.
- **특이 사항**
  - 포켓몬 이름에 포함된 특수문자(예: `-`, `'` 등)는 게임에서 **자동으로 생략**되어 처리됩니다.

---

## 2. 개발 환경

- 언어: Python 3.x
- UI: (예) `tkinter` 기반 GUI
